Политика конфиденциальности

Это приложение «PacketLoss & SpeedScan» может собирать и использовать определенную информацию для улучшения пользовательского опыта и показа рекламы.

1. Собираемая информация

  • Данные об использовании приложения (количество запусков, журналы ошибок и т.д.)
  • IP-адрес (отображается только для диагностики и хранится только на вашем устройстве)
  • Рекламный идентификатор (IDFA)

2. Цель использования

  • Анализ и отображение качества сети
  • Показ рекламы (с использованием Google AdMob)
  • Анализ сбоев и ошибок

3. Внешняя передача данных

Это приложение использует следующие внешние сервисы:

  • Google AdMob (реклама)
  • Cloudflare (тест скорости)

4. Разрешение на отслеживание

При запуске приложения может появиться всплывающее окно с запросом разрешения на отслеживание для рекламных целей. Персонализированная реклама не будет показана без разрешения пользователя.

5. Функция VPN и захват пакетов

Это приложение использует локальный VPN-туннель для диагностики сети. Этот VPN не проходит через внешние серверы, и вся обработка выполняется на вашем устройстве.

  • Захваченные данные пакетов хранятся только на вашем устройстве
  • Данные не передаются на внешние серверы
  • При использовании функции расшифровки HTTPS сертификат CA устанавливается по действию пользователя

6. Подписки

Это приложение предлагает покупки в приложении (подписки). Обработка платежей осуществляется через платежную систему Apple App Store, и мы не собираем и не храним платежную информацию (например, номера кредитных карт).
