Kebijakan Privasi

Aplikasi "PacketLoss & SpeedScan" dapat mengumpulkan dan menggunakan informasi tertentu untuk meningkatkan pengalaman pengguna dan menampilkan iklan.

1. Informasi yang Dikumpulkan

  • Data penggunaan aplikasi (jumlah peluncuran, log kesalahan, dll.)
  • Alamat IP (hanya ditampilkan untuk diagnostik dan disimpan di perangkat)
  • Pengidentifikasi Iklan (IDFA)

2. Tujuan Penggunaan

  • Menganalisis dan menampilkan kualitas komunikasi
  • Menampilkan iklan (menggunakan Google AdMob)
  • Menganalisis kerusakan dan masalah teknis

3. Transmisi Informasi ke Pihak Ketiga

Aplikasi ini menggunakan layanan eksternal berikut:

  • Google AdMob (periklanan)
  • Cloudflare (diagnostik kecepatan)

4. Izin Pelacakan

Saat aplikasi dibuka, mungkin akan muncul pop-up izin pelacakan untuk menampilkan iklan. Tanpa izin, iklan yang dipersonalisasi tidak akan ditampilkan.
