Kebijakan Privasi

Aplikasi ini "PacketLoss & SpeedScan" dapat mengumpulkan dan menggunakan informasi tertentu untuk meningkatkan pengalaman pengguna dan menampilkan iklan.

1. Informasi yang Dikumpulkan

  • Data penggunaan aplikasi (jumlah peluncuran, log kesalahan, dll.)
  • Alamat IP (ditampilkan hanya untuk tujuan diagnostik dan disimpan hanya di perangkat Anda)
  • Pengidentifikasi iklan (IDFA)

2. Tujuan Penggunaan

  • Analisis dan tampilan kualitas jaringan
  • Tampilan iklan (menggunakan Google AdMob)
  • Analisis kerusakan dan kesalahan

3. Transmisi Data Eksternal

Aplikasi ini menggunakan layanan eksternal berikut:

  • Google AdMob (periklanan)
  • Cloudflare (tes kecepatan)

4. Izin Pelacakan

Saat aplikasi diluncurkan, popup izin pelacakan mungkin muncul untuk tujuan periklanan. Iklan yang dipersonalisasi tidak akan ditampilkan tanpa izin pengguna.

5. Fungsi VPN dan Penangkapan Paket

Aplikasi ini menggunakan tunnel VPN lokal untuk diagnostik jaringan. VPN ini tidak melewati server eksternal, dan semua pemrosesan diselesaikan di perangkat Anda.

  • Data paket yang ditangkap disimpan hanya di perangkat Anda
  • Tidak ada data yang dikirim ke server eksternal
  • Saat menggunakan fitur dekripsi HTTPS, sertifikat CA diinstal melalui tindakan pengguna

6. Langganan

Aplikasi ini menawarkan pembelian dalam aplikasi (langganan). Pemrosesan pembayaran dilakukan melalui sistem pembayaran Apple App Store, dan kami tidak mengumpulkan atau menyimpan informasi pembayaran (seperti nomor kartu kredit).
