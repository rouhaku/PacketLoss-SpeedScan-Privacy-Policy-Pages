Политика конфиденциальности

Приложение "PacketLoss & SpeedScan" может собирать и использовать определённую информацию для улучшения пользовательского опыта и отображения рекламы в соответствии с целями, указанными ниже.

1. Собираемая информация

  • Данные об использовании приложения (количество запусков, журналы ошибок и т. д.)
  • IP-адрес (отображается только для диагностики и сохраняется локально на устройстве)
  • Идентификатор рекламы (IDFA)

2. Цели использования

  • Анализ и отображение качества сети
  • Отображение рекламы (с использованием Google AdMob)
  • Диагностика сбоев и других проблем

3. Передача данных третьим лицам

Приложение использует следующие сторонние сервисы:

  • Google AdMob (реклама)
  • Cloudflare (диагностика скорости)

4. Разрешение на отслеживание

При запуске приложения может появиться всплывающее окно с запросом на разрешение отслеживания. Персонализированная реклама не будет отображаться без явного согласия пользователя.
